# Kerala Wedding Photos

This folder contains all Kerala wedding photos used in the customer-facing shared page.

## Folder Structure

```
public/kerala-wedding-photos/
├── albums/          # Album preview images
├── frames/          # Frame product images
└── decor/           # Decor item images
```

## How to Add Images

### Albums Folder (`/albums/`)
Add your album thumbnail and preview images with these naming patterns:

- **Album 1 (Wedding Highlights):**
  - `album1-thumbnail.jpg` - Main thumbnail
  - `album1-1.jpg` through `album1-8.jpg` - Preview images for scrolling

- **Album 2 (Ceremony Collection):**
  - `album2-thumbnail.jpg` - Main thumbnail
  - `album2-1.jpg` through `album2-6.jpg` - Preview images

- **Album 3 (Reception Fun):**
  - `album3-thumbnail.jpg` - Main thumbnail
  - `album3-1.jpg` through `album3-6.jpg` - Preview images

- **Album 4 (Family Portraits):**
  - `album4-thumbnail.jpg` - Main thumbnail
  - `album4-1.jpg` through `album4-5.jpg` - Preview images

### Frames Folder (`/frames/`)
Add your frame product images:

- `frame1.jpg` - Wedding Photo Collage
- `frame2.jpg` - Anniversary Frame
- `frame3.jpg` - Romantic Quote Frame
- `frame4.jpg` - Traditional Kerala Wedding Portrait
- `frame5.jpg` - Black Frame with Gold Mat
- `frame6.jpg` - Intimate Couple Portrait
- `frame7.jpg` - Ceremony Moment Frame
- `frame8.jpg` - Reception Celebration Frame

### Decor Folder (`/decor/`)
Add your decor item images:

- `decor1.jpg` - Wedding Photo Heart Decor
- `decor2.jpg` - Family Gallery Wall
- `decor3.jpg` - Traditional Kerala Canvas
- `decor4.jpg` - Ceremony Frame Set
- `decor5.jpg` - Wedding Memory Board
- `decor6.jpg` - Acrylic Print
- `decor7.jpg` - Traditional Wood Panel
- `decor8.jpg` - Metal Print with Gold Frame

## Supported Image Formats

- `.jpg` / `.jpeg`
- `.png`
- `.webp`

## Image Sizes (Recommended)

- **Thumbnails:** 400x300px or larger
- **Preview Images:** 600x400px or larger
- **Frame/Decor Images:** 600x400px or larger

## Notes

- All images should be optimized for web use
- Use consistent aspect ratios for better display
- Images are automatically optimized by Next.js Image component
- Make sure all filenames match exactly (case-sensitive)
